About Election DApp

This is a web application that allows you to vote for the candidates in the election. You will have to use polygon mumbai testnet to vote.

How to add mumbai polygon testnet to metamask?

  1. Go to mumbai.polygonscan.com
  2. Click on the button "Add mumbai network" at the footer.
